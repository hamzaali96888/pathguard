DROP YOUR .hl7 PATHOLOGY RESULT FILES INTO THIS FOLDER
=======================================================

PathGuard will automatically detect any new files and display
them in the dashboard within a few seconds.

After being read, files are moved to the "processed/" subfolder.

Supported file extensions: .hl7  .HL7  .pit  .PIT

You do NOT need to do anything else — just drop the file here.

For demonstrations, use the "Load Demo Data" button in the
dashboard to populate the inbox with 15 realistic sample results.
